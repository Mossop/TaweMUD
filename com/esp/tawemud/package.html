The base package for the TaweMUD mudlib.

Package Specification

The base classes for the mud are contained here. Noteably the TaweServer and TaweMUD classes that act as the main server and the startup class. Also contained are the classes for zone and world objects as well as the TaweLoader class which allows code rebooting.
